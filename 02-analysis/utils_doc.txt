ALGOTHON ANALYSIS UTILITIES
===========================

Location: analysis/utils.py

DATA CONVENTION
---------------
Functions accept pandas Series, pandas DataFrames, or NumPy arrays. Rows are
time-ordered observations and columns are instruments. One-dimensional arrays
become a single series. Functions do not modify their inputs. Missing values are
allowed; infinite and non-numeric values are rejected.


calculate_returns(prices, *, method="simple", periods=1, dropna=True)
---------------------------------------------------------------------
Input:
  prices       One or more price series.
  method       "simple", "log", or "difference".
  periods      Positive return horizon in observations.
  dropna       Remove rows where every calculated return is missing.

Description:
  Calculates percentage returns, log returns, or absolute price changes.

Output:
  A Series when prices is a Series; otherwise a DataFrame.


rolling_statistics(returns, *, window=20, annualization=250,
                   risk_free_rate=0.0, min_periods=None)
---------------------------------------------------------------------
Input:
  returns          One or more return series.
  window           Rolling lookback length.
  annualization    Observations per year; normally 250 for Algothon.
  risk_free_rate   Annual risk-free rate expressed as a decimal.
  min_periods      Minimum observations required; defaults to window.

Description:
  Calculates rolling arithmetic mean, sample volatility, and annualized Sharpe.
  Sharpe is undefined (NaN) where volatility is effectively zero.

Output:
  Dictionary with DataFrames under "mean", "volatility", and "sharpe".


summary_statistics(data, *, input_type="prices", return_method="simple",
                   annualization=250, risk_free_rate=0.0)
---------------------------------------------------------------------
Input:
  data              Price or return series for one or more instruments.
  input_type        "prices" or "returns".
  return_method     Return calculation used when input_type is "prices".
  annualization     Observations per year.
  risk_free_rate    Annual risk-free rate expressed as a decimal.

Description:
  Produces consistent descriptive and risk statistics by instrument.

Output:
  DataFrame containing count, mean, annualized return, annualized volatility,
  Sharpe, skew, kurtosis, minimum, median, and maximum.


correlation_matrix(data, *, input_type="returns", method="pearson")
---------------------------------------------------------------------
Input:
  data          Price or return series.
  input_type    "prices" or "returns".
  method        "pearson", "spearman", or "kendall".

Description:
  Calculates cross-instrument correlations. Prices are converted to simple
  returns before correlation when input_type is "prices".

Output:
  Square correlation DataFrame.


autocorrelation(data, *, lags=20, squared=False)
-------------------------------------------------
Input:
  data       One or more series, normally returns.
  lags       Maximum positive lag or an explicit sequence of positive lags.
  squared    Analyze squared values to investigate volatility clustering.

Description:
  Calculates each instrument's sample autocorrelation at the requested lags.

Output:
  DataFrame indexed by lag with one column per instrument.


compare_periods(data, periods, *, input_type="prices", annualization=250)
----------------------------------------------------------------------------
Input:
  data              Price or return series.
  periods           Mapping from period name to inclusive (start, end) index
                    labels, e.g. {"train": (0, 499), "test": (500, 749)}.
  input_type        "prices" or "returns".
  annualization     Observations per year.

Description:
  Runs summary_statistics independently over multiple periods to expose regime
  changes and unstable instrument behaviour.

Output:
  DataFrame indexed by period and instrument.


plot_series(data, *, normalize=False, log_scale=False, title=None, ax=None)
----------------------------------------------------------------------------
Input:
  data          One or more series.
  normalize     Rebase each series to 100 at its first valid observation.
  log_scale     Use a logarithmic y-axis.
  title         Optional chart title.
  ax            Optional existing Matplotlib Axes.

Description:
  Plots multiple series on a shared axis. Normalization enables comparison of
  instruments with different price scales.

Output:
  (Figure, Axes) tuple.


plot_series_indicators(prices, *, ema_windows=(10, 30),
                       volatility_window=20, sharpe_window=60,
                       annualization=250, normalize=False, title=None)
----------------------------------------------------------------------------
Input:
  prices              One or more price series.
  ema_windows         Sequence of positive EMA spans; may be empty.
  volatility_window   Rolling volatility lookback.
  sharpe_window       Rolling Sharpe lookback, or None to omit the panel.
  annualization       Observations per year.
  normalize           Rebase prices and EMAs to 100.
  title               Optional chart title.

Description:
  Creates aligned panels for prices and EMAs, annualized rolling volatility,
  and optional rolling Sharpe. Every supplied instrument is included.

Output:
  (Figure, ndarray of Axes) tuple.


plot_correlation_matrix(data, *, input_type="returns", method="pearson",
                        annotate=False, ax=None)
----------------------------------------------------------------------------
Input:
  data          Price or return series.
  input_type    "prices" or "returns".
  method        "pearson", "spearman", or "kendall".
  annotate      Draw numeric values inside cells.
  ax            Optional existing Matplotlib Axes.

Description:
  Calculates and displays a correlation heatmap with a fixed scale from -1 to 1.

Output:
  (correlation DataFrame, Figure, Axes) tuple.


MINIMAL EXAMPLE
---------------
  import pandas as pd
  from analysis.utils import calculate_returns, plot_series_indicators

  prices = pd.read_csv("data/prices.txt", sep=r"\s+", header=None).T
  prices.columns = [f"instrument_{i}" for i in range(prices.shape[1])]

  returns = calculate_returns(prices)
  fig, axes = plot_series_indicators(
      prices[["instrument_0", "instrument_1"]],
      ema_windows=(10, 30),
      volatility_window=20,
      sharpe_window=60,
      normalize=True,
  )
  fig.show()
